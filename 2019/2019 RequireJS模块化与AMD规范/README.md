## 文件说明

- 项目代码是初学JS时写的`Todo`程序，随便处理了下用来测试**Require.js**模块管理和**r.js**打包

- `r.js`用来合并压缩代码的工具

- `build.js`是`r.js`的配置文件

- `./dist`是打包输出目录

## 打包流程

`node r.js -o build.js`，即可按照`build.js`的配置说明，打包到`./dist`目录下
